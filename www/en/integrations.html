Integrations

Acurella’s modern application programming interface (API) is able to synchronize both internal and external services seamlessly. Choose any integration type from those offered by our company or your alternative.

  • Access to the sites and subsystems of the platform 24/7/365

  • Creating an effective ecosystem within the enterprise

  • HTTP API to simplify system integration

Landing
Commander

Landing Commander is a tool that makes it possible to manage, deploy, and monitor ad versions with a single click.

  • A module used to create landing page frameworks
  • Autocomplete of blocks with texts and wildcards
  • Automatic information uploading to the Internet
  • Integration with DDL services
  • Transfer of page properties, l-n-f settings, styles, images and links to the related content
  • A pre-configured toolset for AV testing.
  • Support for Look And Feel which provides means for changing the style of all landing page blocks with one operation.
  • Setting up integration with the payment system and third-party analytics in real time.
  • Dynamic support for multiple languages
  • Automatic transfer of client events to the analytics aggregator.

Integration
with ELS

ELS is a system that can process any real or virtual currency and fully supports any loyalty programs in an automated fashion.

  • An easy exchange of bonuses for tangible goods and services.
  • A current balance in real time.
  • Templates of letters for the loyalty program clients.
  • A configurable access for third-party users.
  • Millions of transactions processed simultaneously.
  • High performance in global marketing campaigns.
  • Guaranteed, timely executed transactions.
  • A multilevel wallet system.
  • Built-in rules for managing bonuses.
  • Rules for automatic write-off upon expiration.

Integration with
the ECM-system

ECM is a scalable system developed to tolerate high loads, that is responsible for any data exchange (including electronic documents) with other systems, clients or employees.

  • An email template module
  • An ability to create and store email templates of different styles and sizes.
  • Adaptation of the email language based on client data in CRM.
  • A module for editing text message templates.
  • Setting up PUSH messages, messages in chats, mobile applications, on websites with wildcard support.
  • Message queue management.
  • A set of dynamic KPIs to avoid massive blocking.
  • A guaranteed message delivery or prompt notification of support services about a malfunction.
  • Sending messages through any messenger with the API integration available.
  • A communication campaign management service.
  • Event campaign launching (for example, birthday greetings).
  • A bot management system.

Integration with
call servers

A solution for integration with call servers that provides extensive real-time reports, ensures quality control and automates some of the managers’ operations, and therefore makes it possible to automatically manage call queues and monitor the quality of operators’ work.

  • A ready-for-service script module
  • Script adaptation for the desired language (DLL)
  • Normalizing unstable audio signals
  • A training module for beginners that preserves confidentiality
  • Editing audio files (for example, noise suppression)
  • A module for dialog moderation by checkpoints
  • Call quality assessment by managers
  • Automated request processing
  • Integration with technical support
  • Converting recordings to multilingual support texts
